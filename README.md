# SC-DS-TASK1

## SkillCraft Technology - Data Science Internship

### Task 01: Population Distribution Visualization

This project was completed as part of the SkillCraft Technology Data Science Internship.

The objective of this task is to create a bar chart or histogram to visualize the distribution of a categorical or continuous variable.

### Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

### Dataset
World Bank Population Dataset

### Project Description
In this project, population data was analyzed using Python. The dataset was loaded and processed using Pandas.

The population data for the year 2022 was selected, and the top 10 most populated countries were identified.

A bar chart was then created using Matplotlib to visualize the population distribution among these countries.

### Files in this Repository
- Population_Distribution_Task1.ipynb → Google Colab notebook containing the code
- population_chart.png → Bar chart visualization
- README.md → Project documentation
